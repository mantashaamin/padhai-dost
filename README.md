# 📚 Padhai Dost
An AI study companion for Class 6-10 Indian students.
Explains NCERT topics in simple Hinglish.

## Why I Built This
My brother (Class 6) and sister (Class 9) struggle with 
English textbooks. This is for them.

## Features
- Hinglish explanations
- Science, Maths, SST support
- Class 6-10 NCERT focused

## Tech Stack
- Python
- Streamlit
- OpenRouter API

## How to Run
pip install -r requirements.txt
streamlit run app.py
